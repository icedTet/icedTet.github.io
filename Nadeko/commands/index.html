Text Channel the command was activated in (Mod)

Command Description
nad play ITEM Will add the ITEM to queue or play it if nothing is playing.
ITEM can be : a Youtube Playlist, a Playlist Key,
 a Youtube Song Link, A Search Term (When using the search, if it doesnt return any verification statement, try appending 'song' to your query
nad vs Votes in Favor of skipping the song. Requires a Majority in the VC
nad queue Gets the Current Queue and displays the first 20.
nad remove NUM Removes the NUMth song in the Queue, or clears the queue if NUM is "all". Will not work if user posesses the 'NoRemoveMusic' role
nad np Gets the currently playing song 
nad loop Toggles Loop Mode ON or OFF (Loop Mode basically just copies the songs to the back of the queue every time they play)
